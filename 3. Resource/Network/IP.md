- [[인터넷 계층]]에서 [[Host]]를 식별하기 위한 식별자
- 버전
	- IPv4
		- 길이 : 32비트
		- 종류
			- Global IP
			- Private IP
			- Loopback
			- Broadcast IP
		- ​- Network ID와 Host ID로 나뉨. NetMask로 Network ID의 길이를 지정
			- IP : 192.168.10.14 
			- NetMask : 255.255.255.0 (24비트)
			- Network ID : 192.168.10.0
			- Host ID : 14
			- CIDR 표기법 : 192.168.10.14/24
	- IPv6 : 128비트
